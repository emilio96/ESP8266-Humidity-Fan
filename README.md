# ESP8266-Humidity-Fan
This is a simple case for a portable led lamp. I've used the following things: 1. 12V 4035 cob led (the one in the photo) 2. 7x18650 batteries connected in parallel 3. 1 x usb-c li-ion battery charger (the one in the photo) 4. 1 x dc-dc step up converter set on 12v output 5. 1 x switch 15x21mm
